🔐 Authentication

GET /auth/github

Initiate GitHub OAuth login. Redirects to GitHub authorization page.

GET /auth/github?cli=true&redirect_uri=http://localhost:8080/callback
GET /auth/github/callback

OAuth callback endpoint. Exchanges code for tokens.

POST /auth/refresh

Refresh access token using refresh token.

{"refresh_token": "string"}
POST /auth/logout

Invalidate refresh token and logout.

GET /auth/whoami

Get current authenticated user information.

📊 Profile APIs

GET /api/profiles/

Get all profiles with filtering, sorting, and pagination.

Query Parameters:

gender
age_group
country_id
min_age
max_age
min_gender_probability
min_country_probability
sort_by
order
page
limit
GET /api/profiles/?gender=male&country_id=NG&min_age=25&sort_by=age&order=desc&page=1&limit=10
GET /api/profiles/search/

Natural language search using plain English.

GET /api/profiles/search/?q=young%20males%20from%20nigeria
GET /api/profiles/export/

Export profiles to CSV (Admin only).

GET /api/profiles/export/?gender=male&country_id=NG
POST /api/profiles/

Create a new profile (Admin only).

{"name": "John Doe"}
GET /api/profiles/{id}/

Get a single profile by UUID.

DELETE /api/profiles/{id}/

Delete a profile (Admin only).
